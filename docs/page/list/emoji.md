# Markdown 表情代码

::: tip
将全部代码中对应 emoji 表情左侧复制后输入你的markdown文本即可显示emoji表情。
相关参考地址：[Emoji](https://github.com/markdown-it/markdown-it-emoji/blob/master/lib/data/full.json)
:::

## 示例

``` md
:100: :1234: :grinning: :smiley: :smile: :grin: :laughing: :satisfied:
```
