# 前端项目模板

# 1. blog github

# 说明

vuepress 技术，vue markdown 静态博客生成器

一个基于 VuePress 构建的用于前端学习、笔记记录、趣文推荐的个人博客网站

# 启动

git clone git@github.com:cjm0/blog.git

cd blog

yarn && yarn dev

# 地址

github 地址 https://github.com/cjm0/blog

项目预览 blog.bigqianduan.top

# 说明

# 启动

cd vssr

# 地址
